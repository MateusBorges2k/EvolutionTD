# 🎮 EvolutionTD

> **The enemy learns. You adapt.**

EvolutionTD is an adaptive Tower Defense game where the enemy analyzes the player's behavior and changes its strategy over time.

This project is being developed from scratch using professional software engineering practices and documented step by step.
